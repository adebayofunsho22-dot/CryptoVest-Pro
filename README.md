# 💎 CryptoVest Pro

A full-stack cryptocurrency investment simulator built with Node.js, Express.js, LowDB, HTML, CSS, and JavaScript.

---

## 🚀 Features

- 👤 User registration and login
- 💰 Virtual investment wallet
- 📈 Live cryptocurrency prices (CoinGecko API)
- 💼 Portfolio tracking
- 📊 Dashboard
- 📱 Responsive interface
- 🔒 Secure password hashing
- 💾 Local database using LowDB

---

## 🛠️ Technologies Used

- Node.js
- Express.js
- JavaScript
- HTML5
- CSS3
- LowDB
- bcryptjs
- Express Session
- CoinGecko API

## ⚙️ Installation

```bash
git clone https://github.com/adebayofunsho22-dot/CryptoVest-Pro.git
cd CryptoVest-Pro
npm install
node server.js
```

Open your browser:

```
http://localhost:3000
```
